Commands Review

  • cd - change directory; cd Documents
  • ls - list
  • pwd - print working directory
  • mkdir - make directory; mkdir my_new_directory
  • rm - remove; rm my_file.txt
  • rmdir - remove directory; rmdir my_new_directory
  • cp - copy a file; cp my_file.txt /home/Documents/other_directory
  • mv - move (or rename); mv my_other_file.txt /home/Documents/other_directory

Get data from the internet

wget <url> Example: text files at Project Gutenberg

Installing things from the internet

  • tar.gz file: tar -zxvf <filename>
  • zip file: unzip <filename>
  • deb file: dpkg -i <filename>

Working with Files

  • touch
  • cat
  • more
  • tail

Search and Find

Use find to find files or directories

  • find -type d
  • find -type f
  • find -name <filename>

Use grep to search in a file

  • grep -n <pattern> <filename>

Other File Commands

  • sed - search and replace; sed s/the/WOW pg84.txt
  • wc - word count tool; wc pg84.txt

Redirecting Output

  • Use > to redirect output; cat > list1
  • Use >> to APPEND output; cat >> list1
  • sort
  • sort < list1 - prints sorted list
  • sort < list1 > sortedlist1 - saves sorted list to new file
  • pipes - redirect commands into another; grep 'the' pg84.txt | wc -w

Command Line Tricks

  • cd -
  • Ctrl-R
  • history
  • !!

Your own commands and Bash

Command ls -a in your home directory to list ALL files, even hidden ones

Open up the file .bashrc in a text editor

"alias" to rename a command or create your own

For changes to take effect, restart terminal or type command "exec bash"

Writing Bash shell scripts

You can create a string of commands to be executed all at once

Open up a text editor and write the commands one after another

Save it as <filename>.sh
